🎮 Application Overview

Deadline Rush is a fast-paced, single-player web game that challenges users to complete dynamically generated tasks before the countdown timer hits zero ⏲️. In addition to the built-in challenges, users can also create and manage their own personalized to-do lists, blending productivity with high-stakes gameplay. It’s a thrilling way to boost focus, test reflexes, and conquer deadlines—all in one experience. ⚡📝

🖥️ Technology Overview

  • Languages: JavaScript (Frontend & Backend), HTML, CSS
  • Libraries/Frameworks: Node.js, Express.js
  • Data Formats: JSON, XML
  • API: RESTful API for task generation and leaderboard updates
  • Web Services: Real-time communication between frontend and backend
  • Security: CORS for secure API requests
  • Version Control: GitHub for collaborative development and source control
  • Hosting: Deployed using Microsoft Azure (App Services) and GitHub integration for continuous deployment
